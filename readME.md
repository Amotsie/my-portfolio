## Installation

### Prerequisites
- Node.js

1. Clone the repository:

```
git clone https://github.com/Amotsie/my-portfolio.git
```

2. Navigate to the project directory:

```
cd my-portfolio
```

3. Install dependencies:

```
npm install
```

4. Start the server:

```
npm run dev
```
